## 📝 Notes

### Color Replacement Guide:
**Find and Replace:**
- `bg-violet-600` → `bg-blue-600`
- `bg-violet-700` → `bg-blue-700`
- `bg-violet-50` → `bg-blue-50`
- `bg-violet-100` → `bg-blue-100`
- `text-violet-600` → `text-blue-600`
- `text-violet-700` → `text-blue-700`
- `hover:bg-violet-` → `hover:bg-blue-`
- `focus:ring-violet-` → `focus:ring-blue-`
- `ring-violet-` → `ring-blue-`
- `border-violet-` → `border-blue-`
- `shadow-violet-` → `shadow-blue-`
- `#9F7AEA` → `#2563eb`
- `#8B5CF6` → `#2563eb`

### Standard DepEd Header Pattern:
```jsx
<div className="bg-white border-b-4 border-blue-600 px-4 md:px-6 py-4 md:py-5">
  <div className="max-w-7xl mx-auto">
    <div className="flex items-center gap-3">
      <div className="h-10 w-10 md:h-12 md:w-12 rounded-md bg-blue-600 flex items-center justify-center shrink-0">
        <svg className="w-6 h-6 md:w-7 md:h-7 text-white" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          {/* Icon path */}
        </svg>
      </div>
      <div>
        <h1 className="text-xl md:text-2xl font-extrabold text-slate-900 uppercase tracking-tight">
          Page Title
        </h1>
        <p className="text-xs md:text-sm font-bold text-blue-700 uppercase tracking-wide mt-0.5">
          Subtitle or Badge
        </p>
      </div>
    </div>
  </div>
</div>
```
