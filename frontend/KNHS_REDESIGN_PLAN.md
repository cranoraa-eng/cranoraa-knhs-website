## 🎨 VISUAL IDENTITY

### Color Palette
```
PRIMARY:
- School Purple: #6B46C1 (official school color - used sparingly as accent)
- Academic Blue: #1E40AF (trust, education, reliability)
- DepEd Blue: #0369A1

SECONDARY:
- White: #FFFFFF (majority of surfaces)
- Light Gray: #F8FAFC (subtle backgrounds)
- Medium Gray: #E2E8F0 (borders, dividers)

ACCENT:
- School Gold: #D97706 (achievements, highlights)
- Success Green: #059669 (attendance, completion)
- Alert Red: #DC2626 (urgent, required)

AVOID: Large purple surfaces, excessive gradients, glassmorphism
```

### Typography
```
HEADINGS: 
- Font-weight: 700-800 (Bold, authoritative)
- Style: Official, academic

BODY:
- Font-weight: 400-500 (Clear, readable)
- Size: 14-16px (accessible)

LABELS:
- Font-weight: 600 (Semibold)
- Transform: Uppercase with tracking
- Style: Official forms/documents
```

### Components
```
CARDS:
- White backgrounds
- Subtle borders
- Minimal shadows
- Academic panel aesthetic
- Clear headers with school context

BUTTONS:
- Solid, professional
- Clear labels
- School color accents
- Official form button style

BADGES:
- Academic indicators
- Subject tags
- Grade levels
- Status markers
```

## ✅ DESIGN PRINCIPLES

1. **Professional & Academic**
   - Formal tone
   - Official documentation style
   - Clear hierarchy

2. **Government Standard**
   - DepEd-inspired
   - Public school aesthetic
   - Institutional trust

3. **School-Centered**
   - Kiwalan NHS identity
   - School colors used as accents
   - Educational focus

4. **Information-Dense**
   - Academic data priority
   - Student metrics
   - School activities
   - No empty space

5. **Accessible & Clear**
   - Readable fonts
   - Clear labels
   - Organized structure
   - Easy navigation
